this is a simple program that documents stuff in journals created by the user. The user can choose either admin or user mode and teir 
expericnce will differ. admin requires full command lines while user will guide the user through the program. The program user can save
their profile to files they create and add posts to that file.

They can change everything about their profile and posts, and can view whats on a file by reading it or printing the contents.
